# SilkDock AI 官网落地页

## 设计需求
对标 OpenRouter / 接口AI 等头部竞品，为 SilkDock AI（基于 LiteLLM 的 AI API 聚合网关）打造高端暗色系落地页。

## 设计风格
- **参考**：Linear.app 深色模式
- **色调**：纯黑/深灰背景 + 蓝紫渐变强调色
- **动效**：微光边框（glow）、渐变文字、数字滚动、无限滚动

## 技术栈
Next.js + TypeScript + Tailwind CSS v4 + shadcn/ui + Magic UI + Framer Motion

## 页面区块
1. Navbar — 毛玻璃导航
2. Hero — 动画网格背景 + 代码预览
3. Metrics — 4 项关键数据 + 滚动计数
4. Features — Bento Grid 6 卡
5. Models — 6 个热门模型卡片（含价格）
6. Getting Started — 3 步上手流程
7. Partners — 无限滚动 Logo 墙
8. CTA — 行动号召
9. Footer — 多栏页脚
